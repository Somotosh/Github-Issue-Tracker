    let = Cannot be declared again, but the value can be changed.
    const =Cannot be declared again and the value cannot be changed.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    map() = changes each element of an array and returns a new array.
    filter() = returns a new array by keeping some elements according to a condition.
    forEach() = only works on each element, does not return a new array.
